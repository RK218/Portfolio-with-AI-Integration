🌐 Personal Portfolio Website

A modern, responsive portfolio website designed to showcase my projects, technical skills, and achievements in an interactive and visually appealing way.

🚀 Overview

This portfolio highlights my work in Artificial Intelligence, Machine Learning, and Software Development while also reflecting my interest in system reliability and production-oriented thinking. The website is built with a focus on clean UI, smooth user experience, and structured presentation of information.

🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Responsive Design Principles

🎨 Design & Development

This project was designed and developed with the help of modern AI-assisted tools to improve efficiency and creativity:

* Claude (code assistance and logic refinement)
* Figma (UI/UX design and layout inspiration)
* ChatGPT (development support, debugging, and optimization)

✨ Features

* Responsive design across devices
* Smooth scrolling and interactive UI
* Dark mode support
* Project showcase with visuals
* Skills section with progress indicators
* Contact section with quick access links

🎯 Purpose

The goal of this portfolio is to present my skills and projects in a professional manner while demonstrating my ability to build, design, and structure real-world applications.
